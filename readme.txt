This is a price comparison website created using MERN stack. Just clone it, install the required dependencies mentioned in the package.json files in the main and sub-directories and run the project.
(This project currenty has a few selected products but could expand to a lot more.)


some notes 
{
to import data from js files-->
cd server
npm run data:import

to delete data -->
cd server
npm run data:destroy
}
